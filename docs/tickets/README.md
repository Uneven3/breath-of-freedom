# Tickets

Trabajo activo: cada ticket abierto es una copia rellenada de
`docs/TICKET-TEMPLATE.md`, nombrada `<slug>.md`. Al cerrarse, el archivo se
elimina y su resumen de una línea pasa a `LOG.md` (el historial completo
queda en git). Ver `docs/WORKTREE-WORKFLOW.md` para pasar de un ticket a un
worktree.
